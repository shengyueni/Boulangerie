# 吗喽的出走

保护好你的猩。

「吗喽的出走」是一个帮助用户记录职场消耗、整理离职想法、准备过渡计划的微信小程序。它不是替用户做决定的工具，而是把混乱的感受整理成记录、趋势和下一步的小型陪伴工具。

当前版本：MVP 0.7.1 角色素材精细化版

## 技术栈

- 微信小程序原生结构
- JavaScript
- WXML
- WXSS
- 本地存储

## 当前功能

- 底部 tabBar 主导航：首页「今日」、日记、出走、百宝箱、心声
- 日记本、新增日记、日记详情
- 事实纪要生成、复制事实纪要、删除日记
- 决心仪表盘统计、原因分布和保护建议
- 泡泡机随机文案和 LD 泡泡点破逻辑
- 愿望清单默认项、新增、完成、删除和本地存储
- 百宝箱工具入口
- 深呼吸练习
- 离职状态自测
- 情绪急救卡片和小动作加入愿望清单
- 心声只读占位页
- 角色陪伴系统第一版：Croissant、Elodie、Gapchick、LD 以轻量陪伴卡进入关键页面
- 角色素材系统精细化：透明背景基础素材 + 13 张状态 / 动作半身素材
- 隐私与安全说明
- 试用反馈问题复制
- 清空本地数据

## 暂不包含

- 登录
- 云开发或后端
- 真实匿名树洞、发帖、评论
- AI 生成或 AI 咨询
- 复杂动画
- 支付
- 医疗诊断、法律咨询、劳动仲裁建议

## 如何运行

1. 打开微信开发者工具。
2. 选择「导入项目」。
3. 项目目录选择本仓库根目录。
4. 使用 project.config.json 中的配置打开项目。
5. 在开发者工具中编译并预览。

## 隐私说明

当前试用版不会上传数据。日记、愿望清单和测试内容保存在当前设备的小程序本地存储中。清空小程序缓存、换设备或在「隐私与安全说明」页清空本地数据后，内容可能无法恢复。

## MVP 0.7 角色陪伴系统第一版

- 新增角色陪伴系统第一版，让 Croissant、Elodie、Gapchick、LD 以轻量陪伴卡进入关键页面。
- Croissant：用户自己的内在小动物 / 当前状态，负责自我陪伴和状态承接。
- Elodie：清醒提醒 / 事实整理，负责写日记、事实纪要和安全边界提示。
- Gapchick：休息恢复 / 生命力重建，负责深呼吸、急救卡和出走后的生活重建。
- LD：职场废话与压迫来源，只作为需要被识别和拆解的职场废话来源出现。
- 当前角色只以本地素材、头像提示和轻量卡片进入页面，不包含角色养成、剧情、语音或动画。
- 当前仍不包含登录、云开发、AI、真实社区、医疗诊断、法律咨询或劳动仲裁建议。
- 数据仍保存在当前设备本地存储中，不上传到后端。
