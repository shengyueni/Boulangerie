# 0.3C 视觉说明

## 视觉方向

0.3C 的目标是让「吗喽的出走」先像一个干净、好读、白底为主的小程序，再在第二眼里看到精致的花纸细节。信息层级永远优先于装饰。

花纸风格不是整屏密集底纹，而是白/米底 + 精致局部装饰。

## 花纸参考图样总结

参考图样共同呈现出剪纸、窗棂、卷草纹、花卉纹、云纹、边框纹和护身符式章牌感。它们的重点不是写实插画，而是对称、克制、细线条和局部装饰。

## 页面背景原则

- 大面积使用白色、奶油白、暖米白。
- 背景只保留极轻纸张感。
- 不使用整页厚重花纹背景。
- 装饰只出现在边角、标题区、卡片边框、章节分隔和底部花边。

## 纹样使用位置原则

- 页面标题区：小印章、细花条、角花。
- 重点卡片：护身符式边框和角花。
- 普通卡片：细边框、浅金棕内线。
- 分组标题：ribbon-title 花条。
- 空状态和提示：保持干净，只用轻虚线边框。

## 色彩系统

基础底色：奶油白、米白、暖白。
文字：深棕、灰棕、深灰。
装饰点缀：靛蓝、石榴橙、豆沙粉、深紫、孔雀绿、少量正红、浅金棕边线。

每个页面尽量只有一个主装饰色，避免平均铺满粉绿黄。

## 卡片系统

- paper-card：基础纸张卡片。
- ornament-card：带花纸边饰的卡片。
- amulet-card：护身符式重点卡片。
- sticker-card：轻贴纸小卡。
- note-card：便签纸条。
- archive-card：档案纸条。
- empty-card：空状态卡。
- ribbon-title：分组标题条。
- tiny-seal：小印章装饰。
- corner-flower：角花装饰。
- floral-divider：花条分隔。

## 按钮系统

主按钮像护身符标签，次级按钮像细边纸签，危险按钮使用深紫红或深棕边线，不使用刺眼大红。小按钮像小印章或小签条。

## 后续角色图接入原则

后续角色图应作为内容主体的补充，而不是压过文本和功能。角色图应预留在标题区、状态卡或模块入口旁的小面积位置，不破坏白底留白和花纸边饰秩序。

## 本轮明确不做

- 不新增功能。
- 不改动数据结构。
- 不引入外部图片或网络资源。
- 不加入角色头像或写实插画。
- 不做复杂动画。
- 不做整屏密集花纹背景。


## 0.3C.1 可见性修复

0.3C.1 将关键花纸装饰从“主要依赖伪元素和很淡的线条”调整为“真实 WXML 节点 + 更清晰的装饰色”。首页、泡泡机、日记、仪表盘、出走计划、心声和工具页都加入了肉眼可见的角花、花纸分隔线、护身符边框、地图路线或纸条边饰。

本次仍坚持：先是干净白底和清楚内容，再在标题、边框、角落和分隔处看到精致花纸细节。

## MVP 0.4 视觉识别度打磨

MVP 0.4 的重点是主要页面视觉识别度打磨和试用前体验收口，不新增功能，不改动核心数据结构，不接入登录、云开发、AI、角色图片或外部网络素材。

本轮视觉原则是：第一眼能看到花纸细节，第二眼仍然觉得页面干净、克制、可读。整体继续保持白底 / 米底小程序气质，接近白底花纸手账、护身符和小王国地图，而不是国潮海报、春节祝福页、满屏底纹、普通粉绿模板或商务蓝白风。

本轮主要落点：

- 首页强化「吗喽王国地图」标题牌、地点入口、小印章、路标和路线感。
- 写日记页强化为「护身符记录纸」，分区卡片像一张张记录纸，选项按钮像小标签，分区之间有可见花纸分隔条。
- 日记列表强化为一组被保存下来的纸条，卡片增加纸张感、角花和信息层级。
- 日记详情强化完整事实纪要为「可复制的证据纸」，复制按钮为印章式按钮。
- 决心仪表盘强化为「Croissant 状态护身符」，保护建议像一张小签文或护身符纸条。

当前版本仍不包含登录、云开发、AI、真实社区、医疗诊断、法律咨询。日记、愿望清单、测试内容和本地心声仍然保存在当前设备本地存储中。
## MVP 0.5 今日职场猩象

「今日职场猩象」是世界观入口，不是真登录页，也不是账号、授权、云端身份或用户系统。它放在首页首屏，用一张今日职场能量卡把用户带入吗喽王国。

视觉关键词：星象卡、花纸边框、护身符、白底 / 米底、细线、角花、小印章。卡片可以有星点、细线圆环和花纸分隔，但仍然保持大留白，不使用外部图片、网络素材、角色头像或满屏底纹。

内容原则：荒谬但清醒，可爱但不轻浮，像运势卡但不是算命。文案应该帮助用户观察职场消耗、整理事实和补现实安全感，而不是制造命运感或催促重大决定。

展示原则：每个指标使用「指标名 + 状态值 + 一句短解释」，避免只给百分比或让用户看不懂指标含义。五个指标固定为耗电量、边界天气、废话云层、顺毛概率、安全感库存。

安全原则：不鼓励冲动离职，不做医疗 / 法律 / 命运判断，不使用命令式重大决策、诊断式结论或命运判断式表达。推荐行动优先引导记录具体事件、拆解模糊评价、照顾身体和检查现实准备项。
