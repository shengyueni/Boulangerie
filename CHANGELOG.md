# CHANGELOG

## mvp-0.8D-bubble-machine-restore-and-ui-polish

- 修复泡泡机定位：从 LD 废话为主的错误方向，恢复为「两个朋友为主、LD 偶尔出现」的小装置。
- 默认进入页面时展示泡泡机主素材和两条朋友泡泡：Gapchick 负责安抚、慢下来和身体安全感；Elodie 负责事实、边界和线索整理。
- LD 泡泡仅在「再吹一次」时低频乱入，概率约 18%，并以较弱的小提示展示，不再承担主视觉或主交互。
- 接入用户提供的 `assets/bubble/bubble-machine.png`，使用 `image mode="aspectFit"`，不拉伸、不裁切、不使用网络图片。
- 泡泡视觉由 WXML / WXSS 绘制，没有新增泡泡图片素材，也没有生成新素材。
- 泡泡机操作改为「再吹一次」「记进日记」「我先收下这两句」。
- 当前仍不做登录、授权、云开发、AI、后端数据库、真实社区、支付、医疗诊断、法律咨询或劳动仲裁建议。

## mvp-0.8C-diary-focus-and-copywriting-polish

- 日记主流程从正面 / 负面记录收敛为离职判断与归因复盘记录。
- 新记录底层继续使用 `type: "negative"` 参与 Croissant 磨损值计算，并新增 `entryKind: "decision_factor"` 作为兼容标记；不迁移、不删除旧数据。
- 旧版 `positive` 记录继续展示为「旧版顺毛记录」，并保留原有 Croissant 顺毛兼容计算。
- 写日记页新增「更想离开什么」和「想靠近什么」两个可选线索字段，事实纪要改为证据式复盘结构。
- 保存页、列表页、详情页、仪表盘和首页快捷入口改为「影响判断的事」「判断记录」「线索整理」语义。
- 文案池按真诚、清醒、有回信感的方向打磨，没有模仿任何具体在世作家的标志性写法。
- 当前仍不引入登录、授权、云开发、AI、后端、真实社区、支付、医疗诊断、法律咨询或劳动仲裁建议，也不鼓励冲动离职。
